About Us

Team bi0s of Amrita Vishwa Vidyapeetham is a well-established group of computer security enthusiasts focusing on various domains such as Web Application Security, Cyber Forensics, Binary Exploitation, Reverse Engineering, Cryptography, Android Security and IoT security. Team bi0s is the No.1 Capture The Flag (CTF) team in India as recognized by CTFTime.org (an international rating for cyber security contestants).

bi0s is largely a CTF team actively taking part in the International CTFs round the year besides carrying out research on security issues in the real world. Fuelled by the urge to do community service, bi0s has been organizing workshops and educating the masses about the need for cyber security.

Event FAQs

What is CTF ??

CTFs are a type of computer security competition. Contestants are presented with a set of challenges which test their creativity, technical (and googling) skills, and problem-solving ability. Challenges usually cover a number of categories, and when solved, each yields a string (called a flag) which is submitted to an online scoring service. CTFs are a great way to learn a wide array of computer security skills in a safe, legal environment, and are hosted and played by many security groups around the world for fun and practice.
