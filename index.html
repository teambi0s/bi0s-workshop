About Us

Team bi0s from Amrita Vishwa Vidyapeetham is a well-established group of computer security enthusiasts focusing on various areas such as Web Application Security, Cyber Forensics, Binary Exploitation, Reverse Engineering, Cryptography Android security and IoT Security.

CTFs (Abbreviation for Capture The Flag) are a type of computer security competition. Contestants are presented with a set of challenges which test their creativity, technical (and googling) skills, and problem-solving ability. Challenges usually cover a number of categories, and when solved, each yields a string (called a flag) which is submitted to an online scoring service. CTFs are a great way to learn a wide array of computer security skills in a safe, legal environment, and are hosted and played by many security groups around the world for fun and practice.

bi0s is largely a CTF team actively taking part in the international CTFs round the year besides carrying out research on security issues in the real world. Fuelled by the urged to do community service, bi0s has been organizing workshops and educating the masses about the need for cyber security.Currently team bi0s is the number 1 Cyber Security team from India as rated by CTFTime

Event FAQs

What is the price of the ticket ?

Early bird price is ₹16,000 (before 7th March). After that it is ₹ 18,000. Lunch and snacks will be served during the workshop.

Participants requirements

Participants should have a basic idea on how a web application works. If you have written a simple web application it is an added advantage.

Who should take this course?

The workshop will be highly useful for developers who want to learn to write secure web applications, pen testers who would like to know how to test web applications, bug bounty hunters, members of Quality Assurance team, team leads and managers who would like to understand the risk of insecure coding and system administrators who want to understand how to harden their resources such as database, web and application servers.

What participants should bring?

It is ideal if participants have a laptop with a Linux distro installed. Participants should have the following softwares pre installed in their laptops:
  • Burp suite
  • Firefox and Chrome
  • Foxy proxy addon installed for both Firefox and Chrome
We will be also providing a Linux VM Image with all softwares installed for those who need it. We shall help in all installations after you have registered so that no time is wasted during training in installation.
